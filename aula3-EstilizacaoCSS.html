Módulo 1: Estilização Profissional e UX

Vamos aprofundar nossas habilidades de estilização para criar aplicações React com aparência profissional e de fácil manutenção. O foco será em criar um sistema de design coeso com variáveis CSS, encapsular estilos com CSS Modules e aprimorar a experiência do usuário com SVGs e páginas de erro 404 bem definidas.

CSS com Variáveis: A Base do seu Design System

Variáveis CSS (Custom Properties) são a espinha dorsal de um design consistente. Ao declarar suas cores, fontes e espaçamentos no seletor :root, você cria uma "única fonte da verdade" para o estilo da sua aplicação. Isso torna a manutenção e a aplicação de temas (como um modo escuro) trivial, pois você só precisa alterar o valor da variável em um único lugar.

CSS Modules: Estilos Encapsulados e Sem Conflitos

Para evitar que estilos de um componente "vazem" e afetem outros, usamos CSS Modules. Ao nomear seu arquivo como Componente.module.css, o sistema de build do React gera nomes de classe únicos em tempo de compilação. Isso garante que seus estilos sejam completamente locais e modulares, eliminando o risco de conflitos de especificidade em aplicações grandes.

SVGs: Gráficos Vetoriais para uma UI Nítida

SVGs (Scalable Vector Graphics) são ideais para ícones e logotipos por serem definidos por vetores, não pixels. Isso significa que eles podem ser redimensionados infinitamente sem perder qualidade. Ferramentas como Vite e Create React App (com configuração adicional) podem importar arquivos .svg diretamente como componentes React, facilitando a estilização e a manipulação via código.
